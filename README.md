# MicroJava Compiler

A compiler for the MicroJava programming language developed as part of the course *Programski prevodioci 1* at the School of Electrical Engineering, University of Belgrade. MicroJava is a simplified Java-like language used for learning the fundamentals of compiler construction.

This project implements the main phases of a compiler, including lexical analysis, syntax analysis, semantic analysis, symbol table management, and bytecode generation for the MicroJava virtual machine. The compiler is implemented in Java using JFlex and CUP.
